# Domain Expert Outreach Playbook

Phase 1 Week 1 onward. Goal: 3 confirmed Bayern-Architekten (plus 1 backup) for Phase 4.5 validation session by Week 6. The lead time for architect availability is 4-6 weeks; this work starts immediately, not in Phase 4.

## Target profile

- Eingetragener Architekt der Bayerischen Architektenkammer.
- 5+ Jahre Bauantrags-Praxis.
- Mix aus residential / commercial / mindestens einer mit Sonderbau-Erfahrung (Versammlungsstätte, Kita, Verkaufsstätte).
- Bayern (Phase 4.5); spätere Phasen adressieren NRW + BW (Phase 18), dann weitere.

## Channels (in priority order)

1. **Warm intros** from existing network. Single best signal. Ask explicitly.
2. **LinkedIn search**: "Architekt Bayern" + Aktivität in den letzten 90 Tagen. Avoid "Tragwerksplaner" - that's secondary ICP.
3. **Kammer-Newsletter** der Bayerischen Architektenkammer (DABonline regional).
4. **Architektenforum** der BAK (online, Münchner regionale Arbeitskreise).
5. **Cold outreach** als letzte Stufe; Conversion <5%.

## Initial message template (DE)

```
Hallo Frau / Herr [Name],

ich entwickle mit Strota ein Werkzeug, das Bauantragsunterlagen vor der Einreichung auf
Vollständigkeit prüft. Bevor wir das Produkt fertigstellen, würde ich gern verstehen,
wie ein Bauantrag in Ihrer Praxis tatsächlich entsteht.

Hätten Sie Lust auf ein bezahltes 90-Minuten-Gespräch (500-700 EUR Honorar) in den
naechsten 4-6 Wochen? Wir gehen einen Ihrer realen Bauantraege durch, ich notiere,
und Sie geben mir ehrliches Feedback. Keine Verpflichtung, keine spaetere Nutzung
ohne Ihre Zustimmung.

Beste Gruesse
[Name]
```

## Session protocol

- 90 Min via Video oder vor Ort (vor Ort bevorzugt).
- NDA vorab gegengezeichnet (`/legal/domain-expert-nda.md`).
- Architekt bringt 5 anonymisierte Vergangenheits-Projekte (alle Verfahrensarten + 1 Sonderbau wenn möglich).
- Strota erzeugt jeden Fall, vergleicht Output.
- Notizen werden im internen Wiki abgelegt (nicht im Repo).
- Honorar wird per ueberweisung am Folge-Werktag ausgezahlt.

## Bus factor

Mindestens 3 aktive Architekten pro Land jederzeit verfuegbar (Risk R12 v5.0). Bei Ausfall sofort Backup aktivieren.
